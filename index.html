Snurra - A JS button spinner

  • Dependency free
  • About 10 KB miminized including css
  • Custom spinners
  • Probably works with frameworks like Bootstrap

Snurra and destroy!

Requires no setup of fancy classes or containers and leaves nothing behind once destroyd.

var btn = document.getElementById('myButton');
var s = Snurra(btn);

/* do something useful */

get('my api call',function(data){

    /* finish up and destroy*/

    s.destroy();
	
    /* all done */
});

Want to do it the hard way?

Pass an element or string as first argument and an object with your options in the other.

var btn = Snurra('#myButton',{
    autoStart: false
});

btn.start();

get('my api call',function(data){
    /* finish up*/

    /* saving your instance for later use */
    btn.stop();
});

How about...

The built in tap-event works well with desktop and touch devices.

Snurra.globals.autoStart = false;
var btn = Snurra('#myButton');
btn.onTap(function(ev){
    get('my api call',function(data){
        btn.stop();
    });
});

Custom SVGs

Add your own image to the stack and use it as a default.

Snurra.addSpinner('/static/bubbles.svg');
Snurra.gobals.maxSize = 40;

Style with CSS

Change colors and more with CSS.

.btn-red .snurra-img {
    fill: yellow
}

Methods

Method Description
stop(function) Starts spinner unless already started. The callback function is optional.
start(function) Stops the spinner unless already stopped. The callback function is optional.
destroy(function) Destroys the Snurra. The callback is optional.
on(event,function,useCapture) Adds a listener to the root-element.
off(event,function,useCapture) Removes a listener to the root-element.
one(event,function,useCapture) Adds a listener to the root-element, once done the listener is removed.
onTap(function) Adds a tap/click listener to the root-element.
tapOff(function) Removes a tap/click listener to the root-element.

Global functions

Function Description
Snurra.addSpinner(url,setAsDefault) Downloads and adds a new spinner image. setAsDefault is optional. Calling this function should be done soon after the page is loaded.

Options

All default options can be changed via Snurra.globals

Snurra.globals.maxSize = 30;

You can also pass the options as parameter at instantiation.

var s = new Snurra(elem,{
    maxSize: 30,
    autoStart: false
});
Property Description
autoStart bool
(default: true) 		Enables/Disabels if the Snurra should start directly once instantiated
autoStartOnTap bool
(default: null) 		Enables/Disabels if the Snurra should start directly once tapped/clicked
onTap bool
(default: null) 		Adds a tap/click listener to the root-element.
maxSize int
(default: 24) 		Max width and height of the image
spinner string
(default: 'default') 		The spinner image. If you want to add your own then use the Snurra.addSpinner(url,true) global function.
